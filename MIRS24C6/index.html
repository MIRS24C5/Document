技術探究報告書

1. 問題の特定

屋外で使用する際において自己位置推定、タスク取得等を行いたい。またMIRSにおいて画像認識をすることは重要なことである。しかし屋外における画像認識は技術的に高度である。 そのため、画像認識ではどのようなものがあるかを調べ、自分たちで期間内に達成できるようなものにすることにした

2. 情報収集と分析

以下の技術を調査した:

  • AIを使った画像認識
  • QRコードを使ったタスク認識
  • カメラを用いずセンサーを使った認識

AIの画像認識は現状では難しいと判断したため、比較的簡単だと思われるQRコードを用いることとした。 またDENSOによる先行例（QRコードを活用し、ロボットの位置補正や作業の切替）があったことも理由の一つである。

3. 仮説の設定

Raspberry Pi Picoを使用することで、以下の成果が期待できると仮定した:

  • モータ制御の応答速度向上
  • 複数モータの同時制御の安定性向上
  • 柔軟な拡張性を活用したシステム拡張

4. 実験・検証

以下の手順で実験を行った:

  1. OpenCVを用いてWEBカメラの映像をRasqberry Piに表示させた
  3. 複数モータを同時制御した際の応答性と負荷状況を測定

シミュレーションにはPythonを使用し、制御アルゴリズムの適正を確認した。

5. 結果の評価

実験結果から以下の知見を得た:

  • Raspberry Pi PicoはArduino UNOに比べ、約30%高速なPWM制御を実現
  • 複数モータ制御時の応答性が大幅に向上
  • 拡張性が高く、周辺機器の追加が容易

一方で、初期設定に時間を要する点が課題として残った。

新たな知見の発見

本技術探究を通じて以下の成果を得た:

  • Raspberry Pi Picoを用いたモータ制御系の有効性を確認
  • システム拡張性の向上により、今後の発展可能性を示唆
  • モータ制御アルゴリズムの最適化手法に関する新たな知見

これらの成果をMIRSMG5標準機に反映し、さらに高性能なモータ制御を実現する技術開発を進める。
